## Prerequisites

- **Docker** >= 20.10
- **Docker Compose** >= 2.0
- **Linux host** with access to `/proc`, `/sys`, and `/dev`
- **lm-sensors** (recommended for temperature metrics)

## Quick Start

```bash
cp .env.example .env
# Edit .env to set GF_SECURITY_ADMIN_PASSWORD
docker compose up -d
```
